Input 输入框

通过鼠标或键盘输入字符

基础用法

<c-input
  placeholder="请输入内容"
  v-model="input">
</c-input>

禁用状态

通过 disabled 属性指定是否禁用 input 组件

<c-input
  placeholder="请输入内容"
  v-model="input1"
  :disabled="true">
</c-input>

带 icon 的输入框

带有图标标记输入类型

可以通过 icon 属性在 input 组件尾部增加显示图标。

<c-input
  placeholder="请选择日期"
  icon-l="success"
  v-model="input2">
</c-input>
<c-input
  placeholder="请选择日期"
  icon-r="plus"
  v-model="input2">
</c-input>
<c-input
  placeholder="请选择日期"
  icon-l="success"
  icon-r="plus"
  v-model="input2">
</c-input>
<c-input
  placeholder="请选择日期"
  icon-l="success"
  icon-r="plus"
  size="sm"
  v-model="input2">
</c-input>
<c-input
  placeholder="请选择日期"
  icon-l="success"
  icon-r="plus"
  size="lg"
  v-model="input2">
</c-input>

文本域

可调整大小，用于输入多行文本信息

通过将 type 属性的值指定为 textarea。

<c-input
  type="textarea"
  v-model="textarea">
</c-input>

复合型输入框

可前置或后置元素，一般为标签或按钮

可通过 slot 来指定在 input 中前置或者后置内容。

<c-input placeholder="请输入内容" v-model="input3">
  <template slot="addon1">Http://</template><template slot="addon2">www</template>
</c-input>
<c-input placeholder="请输入内容" v-model="input4">
  <template slot="addon2">.com</template>
</c-input>
<c-input placeholder="请输入内容" v-model="input4">
  <template slot="btn1">
    <button class="btn" type="button">
     前置
    </button>
    <button class="btn" type="button">
      Options
    </button>
  </template>
  <template slot="btn2">
    <button class="btn" type="button">
      Search
    </button>
    <button class="btn" type="button">
      Options
    </button>
  </template>
</c-input>
<c-input placeholder="请输入内容" v-model="input5" style="width: 300px;">
  <c-select v-model="select" slot="prepend">
    <c-option label="餐厅名" value="1"></c-option>
    <c-option label="订单号" value="2"></c-option>
    <c-option label="用户电话" value="3"></c-option>
  </c-select>
  <c-button slot="append" icon="search"></c-button>
</c-input>

尺寸

可通过 size 属性指定输入框的尺寸，除了默认的大小外，还提供了 large、small 和 mini 三种尺寸。

  <c-input
    inline
    size="lg"
    placeholder="请输入内容"
    v-model="input6">
  </c-input>
  <c-input
    inline
    placeholder="请输入内容"
    v-model="input7">
  </c-input>
  <c-input
    class="col sm6"
    size="sm"
    placeholder="请输入内容"
    v-model="input8">
  </c-input>

带输入建议

根据输入内容提供对应的输入建议

激活即列出输入建议
输入后匹配输入建议

<c-row class="inline-input border-grid">

      v-model="state1"
      :fetch-suggestions="querySearch"
      placeholder="请输入内容"
      @select="handleSelect"
    ></c-autocomplete>

  <c-col :span="12" class="tac">
