代码高亮 代码高亮是通过根据代码语言语法给代码中的单词、字符标记为不同颜色来显示的方法。代码高亮能大大提高代码的查阅体验。在web界面加上代码高亮非常容易，在产品中如需要向用户展现代码应该使用代码高亮。 这里推荐的方案是 Google Code Prettify 实现代码高亮。Google Code Prettify能够自动识别代码语言类型并正确应用语法高亮。
