表单 > 基础表单

为 <input>、<textarea>、<select> 添加 form-ctrl 类即可得到统一外观的表单控件。

所有受支持的表单控件包括：

  • <select class="form-ctrl">
  • <textarea class="form-ctrl">
  • <input type="text" class="form-ctrl">
  • <input type="password" class="form-ctrl">
  • <input type="datetime" class="form-ctrl">
  • <input type="datetime-local" class="form-ctrl">
  • <input type="date" class="form-ctrl">
  • <input type="month" class="form-ctrl">
  • <input type="time" class="form-ctrl">
  • <input type="week" class="form-ctrl">
  • <input type="number" class="form-ctrl">
  • <input type="email" class="form-ctrl">
  • <input type="url" class="form-ctrl">
  • <input type="search" class="form-ctrl">
  • <input type="tel" class="form-ctrl">
  • <input type="color" class="form-ctrl">
